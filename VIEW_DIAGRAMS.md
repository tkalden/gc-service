# How to View Architecture Diagrams

## Option 1: HTML Viewer (Easiest - No Plugins Needed!)

**Just open the HTML file in your browser:**

1. Navigate to: `ARCHITECTURE_DIAGRAMS_VIEWER.html`
2. Double-click the file (or right-click → Open With → Browser)
3. All diagrams will render automatically!

**Supported Browsers:**
- ✅ Chrome
- ✅ Firefox
- ✅ Safari
- ✅ Edge
- ✅ Any modern browser

---

## Option 2: VS Code Extensions

### For VS Code Users:

1. **Install Mermaid Preview Extension:**
   - Open VS Code
   - Go to Extensions (Cmd/Ctrl + Shift + X)
   - Search for: `"Markdown Preview Mermaid Support"`
   - Install by: `bierner.markdown-mermaid`

2. **Or install Mermaid Editor:**
   - Search for: `"Mermaid Editor"`
   - Install by: `tomoyukim.vscode-mermaid-editor`

3. **View the diagrams:**
   - Open `ARCHITECTURE_DIAGRAMS.md`
   - Press `Cmd/Ctrl + Shift + V` to preview
   - Diagrams will render automatically

---

## Option 3: GitHub (If Repository is on GitHub)

1. Push the `ARCHITECTURE_DIAGRAMS.md` file to GitHub
2. GitHub natively supports Mermaid diagrams
3. View directly in the repository

---

## Option 4: Online Mermaid Editors

1. Go to: https://mermaid.live/
2. Copy the Mermaid code from `ARCHITECTURE_DIAGRAMS.md`
3. Paste into the editor
4. View and export as PNG/SVG

---

## Troubleshooting

### Diagrams not showing in HTML?
- Check internet connection (Mermaid loads from CDN)
- Try a different browser
- Clear browser cache

### VS Code not rendering?
- Make sure extension is installed and enabled
- Try reloading VS Code window (Cmd/Ctrl + Shift + P → "Reload Window")

### Need to export as images?
- Use https://mermaid.live/ to export PNG/SVG
- Or use browser screenshot tools
